# 品牌名使用规范(强约束)
brand_name_usage:
  rule: |
    口播台词中尽量避免连着提到「品牌名 + 产品类目」的官方全称。
    可以单独玩品牌名「轻醒」两个字的梗,可以说「这个酸奶」「黄桃味的」,
    但不要把「轻醒酸奶」「轻醒希腊酸奶」这种四字以上全称连在一起念。
  
  rationale: |
    粉丝对「连说品牌全称」高度敏感,一听就是广告。
    博主自己的语言习惯也很少这样说话——她们会说「这个东西」「这罐子」「我最近喝的这个」。
  
  preferred_methods:
    - 通过画面字幕浮出品牌名
    - 通过产品包装特写显示
    - 通过笔记 tag 带出品牌
    - 通过单字玩梗(如「轻醒,名字起得还挺准的」「轻轻醒醒小酸奶」)
  
  acceptable_in_voiceover:
    - 「这个酸奶」
    - 「黄桃口味的」
    - 「我最近发现的这个」
    - 「叫轻醒」(单独提品牌名,不带产品类目)
    - 「这罐子」
  
  forbidden_in_voiceover:
    - 「轻醒酸奶」(品牌+品类连读,广告感强)
    - 「轻醒希腊酸奶」(品牌+品类连读)
    - 「轻醒高蛋白酸奶」(品牌+卖点连读)

# 开场反硬广约束(强约束)
opening_anti_ad_constraint:
  scope: 笔记标题 + 开场 10 秒 + 前 2 个分镜的口播和画面
  
  forbidden:
    - 标题中出现具有产品指向性的代词(如「这勺」「这一杯」「这罐」)
    - 开场口播明确指向产品(如「今天给大家分享一款...」)
    - 前 2 个分镜的画面包含产品近景特写或品牌字样清晰可见的镜头
  
  acceptable_in_opening:
    - 远景出现产品(从冰箱拿出来但拉得够远以至于看不清字)
    - 标题指向场景或状态(「打工人精神好的秘诀」「早起脑爆日」「今天 7 点自己醒了」)
    - 标题用反问句式或情绪表达(「谁不想周一也清爽」「i 人下班日记」)
  
  rationale: |
    小红书用户对「前 3 秒判断是不是广告」的训练程度极高。
    一旦开场识别为广告,完播率立刻崩,后面的种草段再精彩也救不回来。
    产品应该在博主的内容序列「自然进展到那里」时出现,而不是在标题就预告它。
  
  bad_title_examples:
    # 这些标题一看就是硬广
    - "打工人精神好的一天从这勺开始"  # 「这勺」太指向
    - "i 人下班吃这个真的会上瘾"      # 「这个」+「上瘾」组合像广告
    - "早餐囤这一罐就够了"             # 「这一罐」+「囤」是带货语
  
  good_title_examples:
    # 这些标题指向场景或情绪,产品自然嵌入内容里
    - "打工人精神好的秘诀(早起脑爆日)"
    - "今天 7 点自己醒了,还有比这更好的事吗"
    - "i 人下班日记 | 今天躺在地毯上"
    - "早起做了点不一样的事"

## 五、合规约束(强约束,违反即拦截)

```yaml
compliance_constraints:
  hard_red_lines:
    # 广告法 + 食品安全法层面
    - category: 减肥相关
      forbidden_terms:
        - 减肥
        - 瘦身
        - 减脂
        - 燃脂
        - 暴瘦
        - 一周瘦 X 斤
        - 7 天瘦 X 斤
      forbidden_implications:
        - 「我喝了之后瘦了」
        - 「替代正餐控制热量」
        - 「比 X 食物更瘦/更低卡」
        - 任何前后体重对比

    - category: 治疗与降糖
      forbidden_terms:
        - 治疗
        - 治愈
        - 根治
        - 疗效
        - 抗衰老
        - 降糖
        - 降血糖
        - 控糖(用于功效宣称时)
        - 改善 XX 病
      forbidden_implications:
        - 暗示具有医疗保健功能
        - 替代医疗作用

    - category: 广告法极限词
      forbidden_terms:
        - 最
        - 第一
        - 独家
        - 唯一
        - 绝对
        - 100%
        - 永远
        - 顶级
        - 国家级

    - category: BMI 异常 / 极端身材叙事
      forbidden_implications:
        - 不与「81 斤」「BMI <17」「瘦子吃什么」类内容同框
        - 不出现「催吐」「绝食」「断食」相关字眼
        - 不暗示理想身材是极瘦

  soft_caution_zones:
    # 这些用法不直接违法但会增加平台过审风险,使用时必须配「软化措辞」
    - 「饱腹感」
      softening: 「适合早上吃完比较扛饿」(不能说「吃了就不饿了」)
    - 「高蛋白」
      softening: 描述事实可以,但不能加「补充蛋白质」「增肌」等功能性宣称
    - 「低负担」
      softening: 「负担小一点」(不能说「不长胖」「不增加体重」)
```

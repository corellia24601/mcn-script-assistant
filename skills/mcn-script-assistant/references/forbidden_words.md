## 七、品类专属违禁词:酸奶 / 食品

```yaml
category_specific_dairy:
  # 这些是希腊酸奶/低温奶制品类目特殊敏感词
  - 替代正餐
  - 当饭吃
  - 不用吃饭
  - 减肥代餐
  - 抗炎
  - 改善肠胃(超过事实描述)
  - 治便秘
  - 调理肠胃(医疗暗示)
  
  # 软化措辞替换建议
  soft_replacements:
    "饱腹感强":
      replace_with: "适合早上吃完比较扛饿"
    "不长胖":
      replace_with: "负担小一点"
    "高蛋白":
      keep: "高蛋白(描述事实可以)"
      forbidden_extension: "补充蛋白质 / 增肌 / 增加肌肉"
    "0 蔗糖":
      keep: "0 蔗糖(描述配方可以)"
      forbidden_extension: "降糖 / 控糖 / 适合糖尿病"
```

## 九、软化措辞替换库(自动建议)

```yaml
softening_suggestions:
  "效果":
    use: "感受 / 体验"
  "功效":
    use: "特点 / 配方设计"
  "建议":
    use: "建议 / 适合(语境化使用)"
  "100%":
    use: "纯 / 完全"
  "立刻":
    use: "尝一口 / 当下"
  "改善":
    use: "搭配 / 配合"
```

## 十、灰色地带(警告但不拦截,需配软化措辞)

这些词使用时必须配套软化措辞或上下文,否则升级为拦截:

```yaml
caution_zone:
  "饱腹感":
    rule: "必须配「适合早上吃完比较扛饿」式语境,不能孤立使用"
  "蛋白质":
    rule: "可作为成分描述,不能加「补充蛋白质」「增肌」延伸"
  "0 蔗糖":
    rule: "可作为配方描述,不能跟「降糖」「控糖」「适合糖尿病」连用"
  "低负担":
    rule: "可使用,但不能延伸为「不长胖」「不增加体重」"
  "健康":
    rule: "可使用,但不能宣称具体健康效果(如「健康肠道」「健康血糖」)"
```

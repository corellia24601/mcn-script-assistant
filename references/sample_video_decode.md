
## @凑活活 @Catherine是小熊 视频案例拆解

**数据来源：灰豚（2026年5月）**
**案例选择保证5个条件同时满足：视频笔记、对博主的常规内容有代表性、30天内发布、数据相对突出、内含推广**

### @凑活活 ·《老乡,人生可以是星露谷》

#### 笔记链接： http://xhslink.com/o/RxZPzVZ4lm
#### 推广产品：家乐麻酱拌面汁
#### 笔记数据

| 指标 | 数值 |
|---|---|
| 视频时长 | 8 分 37 秒 |
| 预估曝光 / 阅读 | 84.7w / 8.3w |
| 点赞 / 评论 / 收藏 / 分享 | 1.2w / 6,494 / 1,866 / 393 |
| 阅读粉丝比 | 33.06% |
| 评论粉丝比 | 2.58%(显著高于平均水平) |

#### 结构拆解(5 段式 vlog 框架)

凑活活的长 vlog 不是流水账,有清晰的「**意外起点 → 当下美食 → 计划行动 → 落空反转 → 治愈收尾**」五段叙事:

1. **意外起点**(0:00-0:30):快闪铺垫 + 一个小日常意外(本笔记是「平安夜把床弄脏」)
2. **当下美食**(0:30-2:30):聚焦一两样具体的吃喝(米露 + 麻辣烫),包含**完整制作过程 + 第一口反应**——这一段是产品植入的黄金位置,本笔记的家乐麻酱拌面汁就植入在这里
3. **计划行动**(2:30-5:00):换装出门做一件「有仪式感但不重要」的事(模仿星露谷挖野葱)
4. **落空反转**(5:00-7:00):事情没按计划成功,博主用幽默和自嘲消化失败感
5. **治愈收尾**(7:00-8:37):回家做点东西转化「失败」+ 展示一个手作成果

**对脚本的启示**:即使是 60-90 秒的短脚本,也需要保留「意外/小波折 + 当下享受 + 自嘲/松弛收尾」的三段微缩结构,而不是「开场→产品→CTA」的硬广骨架。

#### 开场快闪(0-3 秒钩子模板)

**模板**:`3-5 个跳跃镜头 + 1 句配音「定情绪」`

本笔记 5 个镜头 4 秒内完成,**只是给视频预告今天会发生哪些事**,不交代逻辑、不强行制造冲突——这是凑活活区别于「冲突型钩子博主」的关键。

随后第一句配音是「**昨晚是平安夜,偏偏早上起床的时候把床弄脏了,多么痛的领悟,只有女生能懂**」——用一个**有具体时间锚点的小尴尬事**直接把观众拉进「我也是这样」的共情语境。

**给「轻醒」脚本的复用提示**:
- 不要用「家人们今天给大家分享一款好物」式开场
- 用「3-5 个生活片段快闪」+ 一句「具体小尴尬/小确幸/小日常」做钩子
- 时间锚点关键词(「今天早上」「昨晚」「这周末」)出现在配音第一句

#### 镜头语言(4 种交替使用)

凑活活的视频里 4 种镜头持续交替,任何一种用太久都会切走:

1. **第三视角中景**(30%)——展示博主全身或上半身,适合「介绍今天要做什么」
2. **第一视角主观**(30%)——用于展示手中物品、做饭过程、近距离细节
3. **手部特写**(20%)——揉米露、拌麻辣烫等动作,持续 1-3 秒就切
4. **物品近景**(20%)——产品包装、成品摆盘等,常配「文本框 + 产品名」字幕

#### 表达方式(凑活活语言风格的 5 个识别特征)

1. **品牌玩梗式称呼**——「**并非拉夫劳伦,是宝宝散步(保罗散步)**」「老演员冷冻麻辣烫」「老乡」。她不直接说品牌名,而是用一个谐音/调侃版本,**这种叫法本身就是笑点和记忆点**
2. **博主圈互文**——「这让我想起一位博主,瘦干螂」。她会自然提到其他博主的外号,营造「我们一起看小红书的」社群感
3. **流行语 + 网络热曲**——抖音热门 BGM(「搓搓~搓搓~」「我们不是情侣却要黏在一起」)随情节切入,3-5 秒就切走
4. **具体感官描述**——「带一点淡淡的麦芽香,放进冰箱冷藏很好喝」「软软糯糯的大米」。**她从不说「好吃」「好喝」,只说能让人脑补出味道的具体词**
5. **自嘲 + 松弛**——「计划赶不上变化」「跟装大象一样,一共就 3 步」。**降低观众的心理防线**,让产品出现时不像广告

#### 产品植入位置(本笔记关键参考)

家乐麻酱拌面汁出现在视频 2:00 左右,即「当下美食段」的尾部。具体植入路径:

1. **博主已经在做麻辣烫**——产品出现时博主已经在产品的天然使用场景里
2. **近景特写产品包装** + **「家乐麻酱拌面汁」文本框**(2 秒)
3. **博主自然口播**「这是我最近新发现的一个神器,不用像之前一样自己调味,也不用稀释麻酱,特别方便,直接淋上去就好了」(用「神器」「方便」「直接」3 个具体功能词,不是「绝绝子」)
4. **第一视角拌面动作 + 闻一下表情**(感官反馈)
5. **回到原本的吃饭叙事**——产品没有打断主叙事,而是融进去

**关键洞察**:8 分 37 秒长视频里,产品段只占 20 秒(3.9%)。这跟「轻醒」短脚本的设计直接相关——即使做 60-90 秒短脚本,产品段也不应该膨胀到全长一半,应该控制在 30 秒内,其他时长用博主调性的非广内容补全。

### @Catherine是小熊 ·《谁不想在周一也清爽松软!(早起脑爆日)》

#### 笔记链接：http://xhslink.com/o/84Ihrq91lBN
#### 推广产品：多芬沐浴露
#### 笔记数据

| 指标 | 数值 |
|---|---|
| 视频时长 | 3 分 09 秒 |
| 预估曝光 / 阅读 | 5.0w / 1.4w |
| 点赞 / 评论 / 收藏 / 分享 | 1,248 / 126 / 114 / 18 |
| 阅读曝光比 | 28.05%(高于行业均值,封面/标题钩子有效) |

#### 结构拆解(打工日记三段式)

Catherine 的工位 vlog 有非常工整的「**起床 → 上班 → 下班**」三段式,每段时长大致均等:

1. **起床段**(0:00-1:10):清晨自律行动 + 早餐 + 洗护产品(本笔记的多芬沐浴露植入位置)
2. **上班段**(1:10-2:10):工位日常 + fit check + 会议吐槽
3. **下班段**(2:10-3:09):吃饭/按摩/购物等放松场景 + 回扣早晨的产品

**关键观察**:产品植入在第一段,但在第三段结尾会**回扣一次**(「按摩阿姨夸我是香香软软的小女孩」把早晨用的沐浴露效果落到「松软」标题上)。这是**首尾呼应式的种草结构**,让产品记忆点在视频结束时再被强化一次。

**对「轻醒」脚本的启示**:即使是 60-90 秒短视频,也可以采用「**早晨植入 + 结尾回扣**」结构——比如开头喝一口轻醒,结尾用「今天没像往常那样三点饿到抢饼干」回扣产品的饱腹感卖点。

#### 开场快闪(0-5 秒钩子模板)

**模板**:`5-6 个生活片段快闪 + 网络热门动感 BGM,无配音`

本笔记开场 6 个镜头 4-5 秒切完、纯靠 BGM 节奏带情绪。视频标题 **「谁不想在周一也清爽松软!(早起脑爆日)」** 也是值得拆解的钩子模板:

- **「谁不想 X」反问句式**——天然引发观众自我代入
- **「清爽松软」具体感官词**——比「好状态」「精神」更有画面感
- **「早起脑爆日」自定义词汇**——把「写脑爆材料的早起日」简化成一个不存在的新词,有标签感

#### 镜头语言(Catherine 的特殊偏好)

她比凑活活更**多用前置自拍 + 滤镜/特效**,拍摄风格更「年轻打工妹」:

1. **前置摄像头自拍口播**(40%)——边走边拍上半身、对镜头说话(出现频率最高)
2. **第三视角中景**(25%)——展示穿搭、工位、动作
3. **小窗口画中画**(5%)——本笔记把欧阳春晓的天鹅湖芭蕾视频放在右上角小窗口配自己锻炼的镜头,形成对比笑点
4. **脸部变形滤镜 + 电子配音**(10%)——制造戏剧化夸张效果,降低广告感
5. **工位特写**(20%)——键盘打字、咖啡杯等

#### 表达方式(Catherine 语言风格的 5 个识别特征)

1. **谐音梗 + 自嘲式时尚术语**——「**欧阳春晓 为何:我舞得不美?**」「**真的是太 duang 了**」
2. **职场黑话戏谑**——「**写脑爆材料**」「**左右脑互搏**」「**啥都想深入的摸鱼时刻突然变得十分的神圣**」。她把职场词玩成段子,**这是对「打工人」身份的精确描绘**——不是抱怨,也不是正能量,是「自嘲式接受」
3. **具体感官 + 比喻**——「**像到了海边一样**」「**我现在就是一口水润大柚子**」「**香香软软的小女孩**」。她的种草语言始终用「比喻 + 身体感受」,不直接说产品好用
4. **碎片化吐槽**——「我们楼里的空调真的是太冷了」「忙着啃手都没空打字」。**让产品植入夹在生活吐槽中间**,出现得不突兀
5. **网络热门梗时效性强**——脸部变形滤镜、电子配音特效是当时段流行的内容,**她对小红书/抖音流行内容的复用速度极快**

#### 产品植入路径(本笔记关键参考)

多芬沐浴露植入在 0:30-1:00 之间(起床段中部),具体路径:

1. **博主自然过渡**——「我现在要准备去沐浴」(产品出现前先有「正在做什么」的过渡句)
2. **第一视角展示产品**——拿起沐浴露瓶身,镜头停留 1-2 秒
3. **挤出来的特写镜头**——「是不是很赏心悦目?里面都是双色水润小爆珠」
4. **使用场景 + 感官描述**——「泡泡超级粘密,是海盐和白柚混合到一起的味道,超级清爽超级提神,像到了海边一样」(粘密/清爽/提神/像海边——4 个感官词层层递进)
5. **2 秒海景插入镜头**——具象化「像海边一样」的比喻
6. **效果展示**——「我现在就是一口水润大柚子」+ 镜头展示胳膊光滑皮肤
7. **结尾回扣**(2 分多钟之后)——按摩阿姨说「香香软软的小女孩」,把沐浴露效果落地到日常体验里

**关键洞察**:3 分 09 秒视频里,产品段约 20 秒(占比 10.5%)+ 结尾回扣 5 秒。这是「早晨植入 + 结尾回扣」双重曝光结构的典型案例,直接被借鉴到「轻醒」脚本里。
